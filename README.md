# durian

**Durian gets better the more you use it.** Unlike other agents, `durian` actually learns as it goes - picks up new skills on the fly, keeps getting sharper over time, reminds itself to save important things, looks back at past conversations, and slowly builds a real picture of who you are.

## Install

For Windows
```bash
irm https://raw.githubusercontent.com/gguf-org/durian/main/scripts/install.ps1 | iex
```

For macOS, Linux
```bash
curl -fsSL https://raw.githubusercontent.com/gguf-org/durian/main/scripts/install.sh | bash
```

After installation, simply type
```bash
durian
```
